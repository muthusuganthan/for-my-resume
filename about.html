Meet Muthu Suganthan: Full Stack Developer, Code Crafter, and UI-Obsessed Problem Solver!

Hi, I'm M. Muthu Suganthan — a passionate MERN Stack Developer with a drive to turn ideas into responsive, real-world web applications. My journey began with curiosity and self-learning, and since then, I've built multiple projects that reflect my love for clean UI, efficient backend systems, and creative problem-solving.

I'm currently pursuing a Full Stack Development course at Revamp Academy, where I actively build applications using React.js, Node.js, Express.js, and MongoDB. I enjoy working on both the frontend and backend, and I believe in continuous learning through real projects and challenges.

Outside the classroom, I've built projects like a Tourism Website, E-commerce Frontend, Firebase Contact Form, and a fully functional Authentication Dashboard using Next.js and NextAuth.js. I also enjoy experimenting with UI libraries like shadcn and Magic UI to create clean, reusable components.

Beyond coding, I'm deeply interested in open-source development, problem-solving, and using technology to make life easier for people. I see every bug as an opportunity and every project as a step closer to mastering my craft.

I'm looking forward to joining a team where I can grow, contribute, and collaborate—because I believe even small ideas can lead to big innovations.
